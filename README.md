# Game of Life
這是一個[康威生命遊戲](https://zh.wikipedia.org/wiki/%E5%BA%B7%E5%A8%81%E7%94%9F%E5%91%BD%E6%B8%B8%E6%88%8F)的模擬器，使用Unity開發。

## 使用說明
### 遊戲初始畫面
![](https://i.imgur.com/p3nfy2f.jpg)

- 中央區域為世界地圖，即細胞可繁殖的區塊
- 右上角按鈕區功能分別為：
    - 清空地圖
    - 三種預設的地圖圖樣
    - 一個拉桿，可隨機生成初始地圖
- 右上方拉桿可調控演化速度，一共五段變速
- 右下角按鈕為開始/停止演化過程

### 五種改變地圖的方式
- 按下按鈕①，生成Glider圖
- 按下按鈕②，生成Lightweight圖
- 按下按鈕③，生成Pulsar圖
- 按下三角形按鈕，將展開一個拉桿，可以於地圖上隨機生成覆蓋率4~100%的活細胞
- 玩家直接點選地圖中的細胞，即可切換生/死狀態
